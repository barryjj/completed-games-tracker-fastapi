IGDB / Twitch

Credentials

IGDB is the game database behind Twitch. Register a Confidential app at dev.twitch.tv/console to get a Client ID and Secret. Uses Client Credentials — no user login required.

How It Works

Once configured, adding a game manually gains a live search field backed by IGDB. Select a result and the title is auto-filled along with the IGDB ID, which enables cover art lookups and richer metadata.

  • Typeahead search — search IGDB by title when adding a game manually; results show cover art, platform tags, and release year to help pick the right entry.
  • Cover art — IGDB cover (portrait, ~600×900) written to the artwork table on add, used as a fallback when Steam art is unavailable.
  • IGDB ID — stored on the game record; enables future enrichment (more metadata, platform data) without re-searching.

Token management is automatic — the Twitch bearer token is fetched on first use and refreshed hourly.
