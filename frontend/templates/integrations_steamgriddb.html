SteamGridDB

Credentials

SteamGridDB hosts community-uploaded cover art. The key is used to look up candidate covers when you click "Find cover" on a library entry.

Get yours at steamgriddb.com/profile/preferences/api
{% if current_user.steamgriddb_api_key %} {% endif %}
How it works

Once your API key is saved, the More menu on each library entry's detail pane gains two options:

  • Find vertical cover — picks from 600×900 portrait candidates (library card art).
  • Find horizontal cover — picks from 460×215 landscape candidates (header art).

For Steam entries, the lookup uses the appid directly. For other sources, it falls back to a title search.
